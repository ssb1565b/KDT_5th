2. HTML (Hyper Text Markup Language)

3. 태그 설명

head

우선 head는 메타데이터를 포함하기 위한 태그로 head 먼저 읽고 body나중에 읽는다. 다음은 head의 속성에 대한 내용이다. 

meta : 
페이지 설명,키워드,저자, 화면 크기 등의 정보. 
주로 브라우저가 정보를 받아드릴때 혹은 검색엔지에서 사용함
meta charset =”UTF-8” : 
charset - 해당 문자가 뭘로 구성되어있는지 / UTF-8은 대부분 언어 수용함
meta http=equiv=”X-UA-Compatible” content="IE=edge” :
http=equiv - 어떤 통신을 사용하는지 
content는 마이크로소프트에서 만들어서 edge에서 열어달라고 하는것
body

body태그는 문서의 내용을 담고있다. 최대한 시멘틱 한태그를 사용하여 웹접근성을 높이면 좋다.
h1 태그는 검색엔진이 처음으로 검색하며 문서에 1개만 존재하도록 한다
